SQL

Aan het einde van 2022 kreeg ik via mijn werk de kans om een korte opleiding in MySQL te volgen. Hoewel deze training vooral gericht was op het bekijken en manipuleren van data op de server voor rapportagedoeleinden, heeft dit mijn interesse in programmeren weer aangewakkerd. Hoewel ik op dit moment nog geen concrete projecten kan presenteren, kijk ik vol enthousiasme uit naar verdere verdieping in databases en SQL tijdens mijn opleiding aan Thomas More. Deze kennis zal me in de toekomst ongetwijfeld in staat stellen om meer met deze waardevolle vaardigheden te doen en wellicht boeiende projecten te realiseren.

Python

Na mijn eerste kennismaking met SQL, raakte ik al snel geïnteresseerd in de mogelijkheid om bepaalde rapportagetaken te automatiseren. Mijn zoektocht leidde me tot Python vanwege zijn toegankelijkheid en flexibiliteit. Vol enthousiasme begon ik met Python, maar helaas, de VPN op mijn werk blokkeerde de import van de modules die ik nodig had, wat me tijdelijk tot stilstand bracht. Uiteindelijk lukte het me om de nieuwe versie van het rapport te creëren met behulp van Power BI, maar Python had me een waardevolle vaardigheid bijgebracht: de basisprincipes van programmeren

Dit inzicht wekte al snel mijn verlangen om dieper in dit vakgebied te duiken. Hieronder vind je enkele links naar kleine projecten die ik heb gemaakt met Python.

  • D&D Dice Roller
  • Slot Machine
  • Email validity checker

HTML & CSS

In mijn vrije tijd begon ik een verkenningstocht naar diverse programmeertalen, waarbij ik me verdiepte in gangbare terminologie en naging wat mijn volgende stappen zouden kunnen zijn. Het viel me op dat HTML en CSS essentiële kennis zijn voor elke programmeur, dus besloot ik me hierop te richten. Met behulp van online cursussen op platforms als Sololearn en VDAB kreeg ik al snel de basisprincipes onder de knie.

Naast de oefeningen die ik op GitHub gedeeld heb, is dit mijn allereerste echte project. Ik merk dat mijn eerdere ervaringen nu hun vruchten afwerpen en me goed van pas komen bij dit project.

JavaScript

Na mijn kennismaking met HTML en CSS voelde JavaScript als de volgende logische stap in mijn leerproces. Het opende voor mij een geheel nieuwe wereld van mogelijkheden om boeiende websites te ontwerpen. Helaas heb ik de JavaScript-cursus van VDAB nog niet kunnen voltooien, aangezien het academiejaar aan Thomas More van start ging. Niettemin ben ik vastbesloten om deze cursus op een later moment af te ronden. Mijn enthousiasme voor programmeren groeit met elke stap die ik zet, en ik kijk uit naar de verdere ontwikkeling van mijn vaardigheden in de wereld van webontwikkeling.

C#

C# is de meest recente programmeertaal waarmee ik in aanraking ben gekomen. Gezien Python en JavaScript beide dynamische programmeertalen zijn, was het even wennen om variabelen te declareren in C#, maar het concept van 'static typing' was gelukkig niet vreemd voor me, dankzij mijn eerdere verkenning van verschillende programmeertalen en terminologie. Doordat ik al een stevige basis in programmeren had gelegd, kon ik me tot nu toe concentreren op het aanleren van de syntax en het ontwikkelen van goede programmeergewoontes binnen de context van C#.
